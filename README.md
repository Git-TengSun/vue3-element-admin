# vue3-element-admin

### 项目介绍
1. 在ChatGPT的协助下完成开发
2. 基于 **Vue 3** + **TypeScript** + **Vite** + **Element Plus**
3. 开发环境
```
操作系统： macOS Ventura 13.2
前端环境： Node v18.15.0 (npm v9.5.0)
开发工具： Vsoce
```
